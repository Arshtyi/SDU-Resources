# SDU 资源

-   本项目收集一些 SDU 相关常用资源

## 关于项目的约定

### 命名

-   均以中文为主
-   尽可能在文件名中指明文件特点

### 格式

-   图片的储存主要使用 `.ai+.ai_tex`、 `.svg`或 `.pdf+.pdf_tex` 格式(基于 Inkscape 等软件)
